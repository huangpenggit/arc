我要入驻
设计事业部-工业设计经理

岗位职责：

1、跟进客户需求，准确解读客户需求，进行产品工业设计及独立提案；

2、参与设计研究工作，配合项目负责人把握设计方向和设计风格；

3、配合项目负责人参与项目质量和流程管理；

4、独立完成设计或设计阶段性工作；

5、配合设计总监带领和培养设计团队，进行团队设计培训。

任职要求：

1、本科以上学历，工业设计专业，三年以上工作经验，有团队管理经验优先。

2、掌握工业设计领域趋势，熟悉设计整个流程，设计思维活跃，灵感丰富，对工业产品的审美观符合大众性，行业性，国际性及时代感的要求，能敏锐的洞察吸收新的流行元素，对造型、色彩、工艺、表面处理有充分的感知和把握能力。

3、具备系统设计分析能力，熟练运用设计软件；

4、具备较强的独立提案能力、项目理解能力和客户沟通能力；

5、具备良好的职业道德素质，工作细心，积极主动，稳重有责任心，善于思考及独立解决问题。

设计事业部－工业设计师/实习生

岗位职责：

1、负责产品的外观设计（概念提案手绘、3D建模、效果图）；

2、负责产品工艺文件、配色方案资料的输出制作；

3、负责产品打样期间的外观确认等。

任职要求：

1、工业设计专业或相关设计专业毕业生或大三、大四学生；

2、有较好的手绘表现功底，能够正确表达思路；

3、思维敏捷、领悟能力强、能够较好的理解设计师的设计思路；

4、严谨的工作作风和良好的团队合作意识，较强的主动性及良好的学习能力。

设计事业部－结构工程师

岗位职责：

1、负责PROE 3D建模；

2、负责各类产品结构设计；

3、2D图尺寸标注，BOM制作；

4、项目跟进、模具跟进。

任职要求：

1、机械、模具、材料相关专业专科以上

2、熟悉塑胶模具、熟悉塑胶材料；

3、熟悉五金模具及材料；

4、熟练PROE及CAD等软件操作；

5、有结构设计1年以上经验；

6、能吃苦耐劳，有责任心及上进心。

设计事业部－研发经理

岗位职责：

1、负责智能硬件或智能产品的生产研发，及开发过程中的项目管理，对于确定实施的新产品开发或老产品改良升级工作；

2、负责设计文档等开发资料的管理，同时负责整合、组织、协调各种内外部开发资源；

3、组织协调相关部门研究缺陷改进方案，并最终解决问题；

4、组织协调参与研发的部门、明确各部门研发过程的分工，按计划执行并完成开发任务；

任职要求：

1、理工类相关专业，本科以上学历；5年以上技术或研发经验，有研发经理或总监工作经验优先考虑；

2、具有统筹能力，独立思考及钻研精神、创新精神；有高度的责任心和强烈的事业心；思维敏捷、勤奋踏实。

3、懂结构设计、机械传动等；

设计事业部－品牌设计师

岗位职责：

1、根据客户需求，提供技术支持，制定设计方案，推进项目实施；

2、分析客户品牌定位，组织确定视觉概念方向；

3、负责项目创意设计及进度把控，合理分派任务；

4、执行创意作品的完成，并确保项目提案通过率；

任职要求：

1、正规美术设计院校专科以上学历；视觉传达、平面设计等相关专业；

2、3年以上品牌设计管理相关经验，有知名公司工作经验者优先考虑，

3、熟练使用设计软件，深刻理解品牌设计工作，对色彩、图形、版式把握敏锐

4、具有出色的审美情趣和创新能力；不拘一格，跨行业的设计能力；

5、思维活跃有创新精神，具备较强的团队领导能力、组织协调能力、提案能力。

项目申报部－主管

岗位职责：

1、负责组织编制公司所有关政府项目申报材料，包括可行性报告、项目计划书、任务书等；

2、负责依据项目申报政策、标准与流程，组织完成项目申报；

3、及时了解各行业动态及相关部委的政策，搜集有价值的行业信息及政策信息；

4、负责本部门管理工作

5、领导安排的其他事务。

任职要求：

1、本科及以上学历，3年以上政府专项资金扶持项目全程管控经验；

2、具备政府专项资金扶持项目管理、知识产权与标准管理、资质认定与管理等知识；

3、具备智能制造、新能源等领域政府专项资金扶持项目申报与管理方面的独立申报能力；

4、一定的公关能力、一家的应用写作能力及良好的团队协作能力；

5、工作细致严谨、认真负责；应急能力与工作抗压能力强；保密意识强；

众创空间事业部－招商专员

岗位职责：

1、负责与公司业务相关的市场推广活动策划及组织实施；

2、负责合作伙伴的拓展、推进与关系维护；

3、负责公司业务合作的寻找、跟进、以及一些业务重点项目的谈判；

4、负责客户的开发与渠道拓展，众创空间招商，拟定客户推广计划；

5、负责客户的订单追踪，订单管理与客户关系的维护和管理；

6、执行市场推广方案并跟踪效果，根据市场情况提出产品开发和改进方面的建议；

7、收集、分析相关市场信息、公司竞争对手情况和销售状况，并撰写分析报告 ；
任职要求：

1、市场营销相关专业优先，大专以上学历，三年以上客户管理经验；

2、负责客户的开发与渠道拓展，拟定客户推广计划，有一定的人脉资源；

3、有大型园区、孵化器、众创空间及相关招商经验者优先

4、会开车，对重庆地形熟悉；

乐创网络－产品助理

岗位职责：

1、协助完成产品调研、产品研发、产品设计以及运营支撑工作；

2、负责产品原型、流程及用户体验、需求方案设计；

3、协调研发、美工、测试进行产品业务流程、开发、测试等；

4、根据产品需求，对产品的整体美术风格、交互设计、界面结构、操作流程等做出设计；

5、用户分析、数据分析、交互设计、优化等方案制作；

6、配合产品经理完成相关任务目标。

任职要求：

1、专科及以上学历，计算机、设计等相关专业毕业，互联网产品工作经验1年以上；

2、熟练使用Axure、Viso、mingmanager、Mockups、Photoshop等软件；

3、熟练掌握交互设计技巧，了解交互设计对用户操作行为的影响；

4、具备良好的逻辑性，对页面的功能布局有较好的把握；

5、会使用相关操作软件，并编写产品需求文档。

如果期望加入，请将个人简历及设计作品（20M以内）发至邮箱：hr@artopcq.com
