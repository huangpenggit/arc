3.登录完Twitter之后，我在JRebel官网上点开Twitter完成登录授权。完成一系列的信息（主要是邮箱账号，其他姓名公司都可以瞎填，电话号码也不需要发送验证码，所以也可以瞎填），填写后会发送一份邮件到你注册的邮箱。注意查收，如果没收到也可能被自动放到邮箱垃圾箱里了。

4.点开邮箱发型果然立马收到了链接

5.点开链接之后，进入到激活页面。②注意是电话号码，经我验证不需要发送验证码，所以不需要填写真实手机号。到了③就会给出激活码了。

6.复制方框中的激活码。

7.填入到IDEA里面。

8.激活成功后如图所示。成功后会挑战到链接https://zeroturnaround.com/software/jrebel/quickstart/intellij/?run=ide#!/project-configuration这是JRebel的使用教程。
